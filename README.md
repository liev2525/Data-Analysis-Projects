# DATA ANALYSIS PORTFOLIO

Hello, and thanks for stopping by! This repository contains personal data analysis projects that I have done utilizing Python, PySpark, SQL, Spark SQL, Excel and Tableau. Below are some of my highlighted projects, including data visualization, statistical analysis, and data-driven insights.

For these projects, I start by conducting most of my analysis using various environments such as MySQL, Databricks, RStudio or Jupyter Notebook in order to replicate the analysis in a database environment. Ultimately, I visualized the results and findings by creating a Tableau dashboard.

These projects reflect my skills in handling diverse datasets and extracting meaningful conclusions to support decision-making. Feel free to check them out and reach out for any questions or feedback.

## [SALARIES ANALYSIS](https://github.com/liev2525/Data-Analysis-Portfolio/tree/main/Salaries%20Analysis)
- Brief overview: Examine how compensation varies by job title, experience level, company size, and location to uncover trends, pay gaps, and high-earning roles

- Final results: [Salary Analysis results](https://www.elizabethelizondo.net/portfolio-brief/salary-analysis)
